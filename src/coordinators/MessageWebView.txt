This MessageWebView is responsible for delegating work between objects: 

 * The setWebViewClient method receives requests from:
-	MessageWebView
-	QuotedMessageMvpView

 * and delegate the request to: 
-	AttachmentResolver
-	OnPageFinishedListener



 * The displayHtmlContentWithInlineAttachments method receives requests from:
-	MessageWebView
-	MessageContainerView
-	QuotedMessageMvpView

 * and delegate the request to: 
-	AttachmentResolver
-	OnPageFinishedListener
