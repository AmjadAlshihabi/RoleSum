This AttachmentPresenter is responsible for delegating work between objects: 

 * The onRestoreInstanceState method receives requests from:
-	MessageCompose
-	AttachmentPresenter
-	RecipientPresenter
-	AccountSetupBasics
-	AccountSetupOutgoing
-	QuotedMessagePresenter
-	FoldableLinearLayout
-	MessageHeader

 * and delegate the request to: 
-	AttachmentMvpView
-	WaitingAction



 * The loadNonInlineAttachments method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	MessageViewInfo



 * The initAttachmentContentLoader method receives requests from:
-	AttachmentPresenter

 * and delegate the request to: 
-	Attachment



 * The attachmentProgressDialogCancelled method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	WaitingAction



 * The addAttachment method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	Attachment
-	Attachment
-	AttachmentViewInfo



 * The performStalledAction method receives requests from:
-	AttachmentPresenter

 * and delegate the request to: 
-	WaitingAction
-	AttachmentMvpView
-	WaitingAction



 * The onClickRemoveAttachment method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	Attachment
-	AttachmentMvpView
-	AttachmentsChangedListener



 * The processMessageToForward method receives requests from:
-	MessageCompose
-	AttachmentPresenter
-	QuotedMessagePresenter

 * and delegate the request to: 
-	AttachmentMvpView
-	MessageViewInfo



 * The initAttachmentInfoLoader method receives requests from:
-	AttachmentPresenter

 * and delegate the request to: 
-	Attachment



 * The processMessageToForwardAsAttachment method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	LocalMessage
-	MessageReference
-	AttachmentMvpView
-	MessageViewInfo



 * The checkOkForSendingOrDraftSaving method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	AttachmentMvpView
-	WaitingAction



 * The onClickAddAttachment method receives requests from:
-	MessageCompose
-	AttachmentPresenter

 * and delegate the request to: 
-	AttachmentMvpView
-	RecipientPresenter



 * The onSaveInstanceState method receives requests from:
-	MessageCompose
-	AttachmentPresenter
-	RecipientPresenter
-	AccountSetupBasics
-	AccountSetupComposition
-	AccountSetupIncoming
-	AccountSetupOutgoing
-	QuotedMessagePresenter
-	FoldableLinearLayout
-	MessageHeader

 * and delegate the request to: 
-	WaitingAction



 * The addAttachmentAndStartLoader method receives requests from:
-	AttachmentPresenter

 * and delegate the request to: 
-	AttachmentMvpView
-	AttachmentsChangedListener
-	Attachment
